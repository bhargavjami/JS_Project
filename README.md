# Egg+ Mini Language Interpreter

This project is a small interpreter inspired by the Egg language from the book *Eloquent JavaScript*.
It can parse and execute a simple expression-based programming language.

## Features

- Variables and functions
- If and while control flow
- Closures and scope
- Arrays support
- Comments using #
- set keyword for updating variables

## Run

```
npm start
```
